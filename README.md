🗂️ Flashcards :) — Study Card Generator
A lightweight, modern, and functional web tool for creating, managing, and printing personalized flashcards. Designed for students looking to optimize their study time with a system ready to print in A4 format.

✨ Main Features
🎨 Adaptive Interface: Clean design with integrated Dark Mode for late-night study sessions.

🖨️ Print Optimization: Automatically generates an A4 PDF with a layout of 8 cards per page (front and back aligned for easy cutting).

💾 Local Persistence: Your cards are automatically saved in the browser (LocalStorage), so you won’t lose anything when closing the tab.

📤 Import/Export: Save your decks as .json files to share or back them up.

⚡ Quick Editing: Intuitive system to add, edit, delete, and shuffle your cards.

📏 Content Control: Character counters to ensure text remains legible once printed.

🚀 Getting Started
Clone the repository:
git clone https://github.com/eemibarahona/flashcards-creator

Open the file:
Simply open flashcards_pro.html in any modern browser. No server or external dependencies required.

📖 Instructions for Use
1. Creating Cards
Enter the Question (front) and the Answer (back) in the text fields. Click "Add Card." The system will alert you if the text is too long for physical printing.

2. Deck Management

Shuffle: Use the "Shuffle" button to change the card order and avoid sequence-based learning.

Edit: If you made a mistake, click the "Edit" button on the corresponding card to reload it into the form.

3. Printing (Critical Step)
To get the best results when printing:

Click the "Print PDF" button.

In the browser's print settings:

Destination: Save as PDF or select your printer.

Margins: None (Very important for alignment).

Background Graphics: Enabled.

When printing double-sided, the answers will align exactly behind their respective questions.
